# Neon Stickman: Stick War - Game Builds

These are the original build files for the game.

| File | Size | Description |
|------|------|-------------|
| `NeonStickWar-release.apk` | 6.5 MB | Android APK — Install on any Android device to play |
| `NeonStickWar-release.aab` | 6.2 MB | Android App Bundle — Upload to Google Play Console for store listing |
| `NeonStickWar-mobile-publish.zip` | 12 MB | Full source project — Next.js + Capacitor project for rebuilding |

## How to Use

- **APK** → Share directly with testers or sideload on Android devices
- **AAB** → Upload to [Google Play Console](https://play.google.com/console) for Play Store distribution
- **ZIP** → Contains the full project source code for making changes and rebuilding

## Important Files (from original extraction)

- `release-keystore.jks` — Signing key for APK/AAB (in upload/NeonStickWar-extracted/)
- `keystore.properties` — Keystore password config (in upload/NeonStickWar-extracted/)

⚠️ **Keep the keystore safe!** If you lose it, you can't update your app on Google Play Store.
